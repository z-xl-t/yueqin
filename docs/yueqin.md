# 民乐月琴

民乐月琴的定弦是固定音高定弦法，栓四根弦。

怀抱月琴，呈演奏姿态，靠近右手边的为四弦、依次向左数为三弦、二弦和一弦。

定音为 g3（四弦）、 d4（三弦）、 g4（二弦）、 d5（一弦）。

基于此定音，可以进行变调，只需要改变月琴按品的指法和对应的唱名（或简谱名）即可。

月琴中常见的自然大调为 1=C 调，1=D 调，1=bE 调，1=F 调，1=G 调，1=C调，1=bB 调。

根据基准调，结合十二平均律，可以推算出其他唱名。

十二平均律作用在一个八度中，即为 **全全半全全全半**。

假设基准调为 1=C4 中音, 那么 G3 为低音4， D4 为中音2，G4 为中音4， D5 为高音2，依次类推。
