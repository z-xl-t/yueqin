## 标准 A 音

标准 A 音的频率是 440Hz 。


## 中央 C 音

比较靠谱的说法：

* 乐音体系中有 97 个音（从最低音 16Hz 到最高音 4186Hz ），而小字一组的 C 音位于这 97 个音的最中间也就是第 49 个，刚好是正中间，所以叫中央 C 音。

## 八度与十二平均律

八度：一个音的频率是另一个音的两倍，他们之间的跨度叫做“八度”。
十二平均律：将一个八度“近似等距离”分成十二份，音高过渡平滑，转调时音程之间比较和谐。
